API Reference

Build custom integrations with the ForgeFlow API. Our REST API follows standard conventions and returns JSON responses.

GET https://api.forgeflow.io/v1/inventory { "items": [ { "id": "inv_123abc", "sku": "RAW-STEEL-001", "name": "Steel Sheet 4x8", "quantity": 245, "unit": "sheets", "updated_at": "2026-03-19T10:30:00Z" } ], "total": 1, "page": 1 }

1 Authentication

All API requests require authentication using an API key. Include your API key in the request header:

Authorization: Bearer YOUR_API_KEY

Generate API keys in Settings → API Keys in your ForgeFlow dashboard.

2 Inventory Endpoints

GET /v1/inventory

List all inventory items. Supports pagination and filtering.

GET /v1/inventory/:id

Get a single inventory item by ID.

POST /v1/inventory

Create a new inventory item.

PUT /v1/inventory/:id

Update an existing inventory item.

DELETE /v1/inventory/:id

Delete an inventory item.

3 Error Codes

Code Description
400 Bad Request - Invalid parameters
401 Unauthorized - Invalid or missing API key
404 Not Found - Resource doesn't exist
429 Rate Limited - Too many requests
500 Server Error - Something went wrong on our end
